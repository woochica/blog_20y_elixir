# Install

``` shell
brew install hugo
git submodule init
git submodule update
make run
```

# Used tools

- Engine: [Hugo](https://gohugo.io/content-management/)
- Hugo theme: [cupper-hugo-theme](https://github.com/zwbetz-gh/cupper-hugo-theme)
